# Research Synthesis

This fixture exercises a source-backed draft for research-style material.

## Key Findings

The pilot runners consistently return valid patch JSON when the request is kept
local, source-backed, and patch-first.

## Confidence Gaps

Direct evidence quality still varies across providers, especially when the
prompt leaves too much room for generic scaffolding.

## Follow-Up Questions
